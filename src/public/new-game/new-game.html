To create a new game do the following:

1. enter to: http://remix.ethereum.org

2. upload all the files in remix folder. all files end with .sol

3. compile and deploy the Quartet.sol contract

4. wait for the contract to be fully deployed and then remember its address

5. enter to the join-a-game section in this site and enter the contract adress from previous instruction

6. invite friends to join your game
